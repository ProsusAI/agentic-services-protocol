# Roadmap

Planned future additions to ASP.

## Near-term

- **Multi-provider carts** — Support checkout across multiple providers in a single session

## Medium-term

- **Additional domain profiles** — Groceries, laundry, pharmacy, home services
- **Subscription management** — Recurring orders and membership tier management
- **Group ordering** — Multiple users contributing to a single cart

## Long-term

- **Cross-marketplace orchestration** — Agent coordinates across multiple ASP-compliant marketplaces
- **Dispute resolution** — Standardized refund/complaint flow
- **Provider onboarding** — Self-service schema validation and capability testing tools
